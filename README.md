# Logistics of Relief: A Statistical Analysis of Gaza Aid Supply & Dispatch Patterns 

Overview of this project:

The main objective of this project is to implement statistical methods and computational tools to 
solve real-world data science problems using an original dataset. In this regard, we have chosen a 
dataset named "State of Palestine: Gaza Supplies and Dispatch Tracking." The dataset was taken 
from the Humanitarian Data Exchange (HDX), managed by the United Nations Office for the 
Coordination of Humanitarian Affairs (OCHA). 

This dataset primarily explains from where the relief supplies came from, how many trucks were 
launched to the war-torn nation of Palestine, and what kinds of products were delivered (such as 
food items, tents, hygiene kits, etc.). 

We, as a team, have studied the records and went through the data and tried to figure out the 
hidden decisions behind the entries. First of all, we have cleaned the dataset and then observed if 
there are any null values or duplicacies that exist or not. Solved those issues and figured out 
which kinds of data are mostly available? Numerical or categorical? Then began using Python's 
statistical tools.
